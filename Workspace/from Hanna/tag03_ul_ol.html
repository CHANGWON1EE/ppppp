목록 태그(ul(순서가 없는 목록 태그), ol(순서가 있는 목록 태그), li)

   ul의 type 속성 : disc(까만 동그라미), circle (하얀 동그라미), square(까만 네모)
   ul의 타입 특징 : 순번의 모양기준이 바뀜
   
   ol의 type 속성 : 1, a, A, i, I
   ol의 타입 특징 : 순번의 기준이 숫자, 소문자, 대문자 등으로 바뀜

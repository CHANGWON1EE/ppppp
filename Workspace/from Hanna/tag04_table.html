표만들기

   table태그 : 표의 시작과 끝
   tr태그 : 행 추가
   td태그 : 열 추가 (데이터 세팅)
   th태그 : 테이블 헤더. 라는 뜻으로 보통 엑셀최상단 데이터컬럼명 쓸때 쓰는 용도
   colgroup태그 : 행을 그룹으로 만든다.
   rowgroup태그 : 열을 그룹으로 만든다.
   caption태그 : 표의 제목
   
   thead : 테이블 대가리
   tbody : 테이블 몸통
   tfoot : 테이블 밑바닥
   
   속성
   width : 수평길이
   height : 수직길이
   border : 테두리 굵기
   cellspacing : td간의 간격을 조절해줌.
   
   span : 인라인 (띄어쓰기 없음)
     ㄴ span="2" 라면 해당방면으로 1칸 더 포함한다. 라는 의미다.
     ㄴ 예시1로서 태그 내부에  등으로도 쓰이고
     ㄴ 예시2로서 태그 내부에
      등으로도 쓰인다.
           
   div : 블록 (띄어쓰기 존재)
